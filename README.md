# Анюткин сад demo

Статический демо-прототип нового сайта и адаптивной админки для цветочного хозяйства.

## Что внутри

- Витрина с реальными товарами, категориями, ценами, остатками и фото из текущего каталога.
- Интерактивная корзина, фильтры и поиск по товарам.
- Админка `#/admin`: продажи, заказы, остатки, товары, очередь СДЭК и чек-лист упаковки.
- Папка `docs/` готова для GitHub Pages.

## Команды

```bash
npm install
npm run dev
npm run pages
```

## Демо-роуты

- Витрина: `/`
- Админка: `/#/admin`
