# Анюткин сад

Готовый статический сайт магазина растений и адаптивная панель управления для цветочного хозяйства.

## Что внутри

- Витрина с реальными товарами, категориями, ценами, остатками и фото из текущего каталога.
- Каталог с поиском, фильтрами, карточками растений и корзиной.
- Страницы `#/catalog`, `#/delivery`, `#/about` и карточки товаров `#/product/:id`.
- Панель управления `#/admin`: продажи, заказы, остатки, товары, очередь СДЭК и чек-лист упаковки.
- Папка `docs/` готова для GitHub Pages.

## Команды

```bash
npm install
npm run dev
npm run pages
```

## Роуты

- Магазин: `/`
- Каталог: `/#/catalog`
- Доставка: `/#/delivery`
- О хозяйстве: `/#/about`
- Панель управления: `/#/admin`
